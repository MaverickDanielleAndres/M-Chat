# database-skill


## Database Migration Rules

- **CRITICAL**: Before creating any migration, scan the codebase (both frontend and backend) to avoid errors and conflicts.
- **CRITICAL**: Create the migration file locally FIRST before applying it.
- **CRITICAL**: Apply the migration using the Supabase MCP server.
